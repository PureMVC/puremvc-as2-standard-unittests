public function testMacroCommandExecute():Void

Tests operation of a MacroCommand. This test creates a new Notification, adding a MacroCommandTestVO as the body. It then creates a MockMacroCommand and invokes its execute method, passing in the Notification. The MockMacroCommand has defined an MockMacroCommand.initializeMacroCommand method, which is called automatically by its constructor. In this method the MockMacroCommand adds 2 MockSimpleCommand objects to itself. The MacroCommandTestVO has 2 result properties, one is set by one MockSimpleCommand multiplying the input property by 2, and the other is set by the other MockSimpleCommand multiplying the input property by itself. Success is determined by evaluating the 2 result properties on the MacroCommandTestVO that was passed to the MockMacroCommand on the Notification body.
